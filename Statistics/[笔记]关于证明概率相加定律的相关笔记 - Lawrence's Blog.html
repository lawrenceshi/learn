其中我认识了概率的相加定律。

我们目前用P（N）表示N发生的概率 U代表加法
我们列举了一些实例方便理解：

#这里是指 仍中1 OR 2 OR 3（N）的概率均为⅙
P( 1 ) = ⅙
P(2) = ⅙
P(6) = ⅙

而概率相加理论则可以写作

P(AUB)=P(A)+P(B)

在概率相加概率当中，我们需要注意几点：

  • 在概率相加概率中，两个概率必须是互不相容的：即为两个概率中不能同时出现相同的数字，例如A = 1 U 2 B = 2 U 3A和B就不互不相容，因为A和B这两个概率中同时包含'2。同理：A = 1 U 2 B = 3 U 4A和B就互不相容，因为A和B这两个概率中没有同时存在的数。

    • 以上的证明方法：因为A = 1 U 2 B = 2 U 3实际计算下来的值是A=⅓,则B=⅓实际算下来的概率是⅓+⅓=⅔，而通过概率相加定律计算出来的结果为P(AUB)=P(1U2U3)=P(1)+P(2)+P(3)=½故此需要互不相容

  • 在几个事件构成完备事件组（我理解为了已经掌握了所有发生的情况）

    • 则：P(A1+A2+...+An)=1
      我们举个栗子来说明：

      乐乐每天都睡觉，有时候在10：00之前，有时候在10：00之后
      假设P( 乐乐每天十点之前睡觉 ) = 300/366和P( 乐乐每天十点之后睡觉 ) = 66/366
      那么 P( 乐乐每天睡觉 ) = P( 乐乐每天十点之前睡觉 ) + P( 乐乐每天十点之后睡觉 ) 则
      P(乐乐每天睡觉) = 1
